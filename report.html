Software Engineering Project Report

Project Title: AI Resume Analyzer System (Powered by Google Gemini)

1. Problem Statement

In today's highly competitive job market, recruiters receive hundreds of applications for a single role, making manual screening time-consuming and prone to human bias. Consequently, many qualified candidates are filtered out by traditional Applicant Tracking Systems (ATS) because their resumes lack exact-match keywords.

Job seekers lack accessible, intelligent tools to evaluate how well their resumes are written. There is a critical need for an automated, context-aware system that can parse, analyze, and score a resume while providing actionable, personalized improvement tips to the candidate.

2. Research Findings & Analysis

Existing Systems vs. Proposed AI System

  • Existing Solutions: Rely heavily on basic keyword matching. They fail to understand semantic meaning, context of achievements, or abstract soft skills.
  • Our Proposed Solution: Leverages Large Language Models (LLMs) like Google Gemini. It reads the resume like a human recruiter, understanding the depth of experience and returning a precise score with dynamic tips.

Feasibility Study

  • Technical Feasibility: Achievable using modern web tech (HTML/CSS/JS) for a responsive front-end and lightweight API calls to Gemini.
  • Operational Feasibility: Drop-zone UI requires zero user training. Logic can easily be adapted for No-Code tools like Glide/Bubble.
  • Economic Feasibility: Cloud-based LLM APIs offer cost-effective, pay-as-you-go infrastructure.

4. Testing and Results

4.1 Testing Methodologies

  • Unit Testing: Tested the file upload component continuously. Ensures only .pdf, .doc, and .docx formats bypass validation.
  • Integration Testing: Verified the transition states (Upload Screen → Loading Screen → Results Dashboard). Ensured timeout functions simulate the API wait accurately.
  • UI/UX Testing: Validated the mathematical logic behind the Circular Progress Bar to ensure the CSS conic-gradient exactly matches the final score percentage. Assessed responsiveness across desktop and mobile views.

4.2 Conclusion & Results

The AI Resume Analyzer successfully demonstrates a paradigm shift from rigid keyword-based ATS systems to intelligent evaluation. The web interface provides a seamless experience, successfully transitioning between file upload, processing logic, and the final animated dashboard. The structured approach allows easy future integration with live APIs or no-code backends, effectively solving the stated problem.
